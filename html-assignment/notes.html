HTML Notes

These are some basic HTML tags I have learned and their meanings.

Description list of HTML tags

<img>
Displays an image on the page. It uses the src attribute for the image path and alt for alternative text.
<a>
Creates a hyperlink to another page or resource. The href attribute defines the link destination.
<header>
Represents the introductory content of a page or section, often containing a logo, heading, or navigation.
<main>
Represents the main content of the document, unique to that page.
<section>
Groups related content into a thematic block and usually includes a heading inside.
